About

This simulator models the Border Gateway Protocol (BGP) using real-world AS relationship data from CAIDA. It implements the Gao-Rexford model — routes from customers are preferred over peers, which are preferred over providers.

ROV (Route Origin Validation) can be enabled per-AS to simulate RPKI deployment. Marked routes are dropped by ROV-enabled ASes.

The full simulation runs in WebAssembly compiled from C++, with zero server-side processing.
